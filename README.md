# doorflow API SDK for typescript

Official typescript SDK for the doorflow API.

## Installation

```bash
npm install @doorflow/doorflow-api
# or
yarn add @doorflow/doorflow-api
```

## Usage

```typescript
import { DoorFlowApi } from '@doorflow/doorflow-api';

const client = new DoorFlowApi({
  clientId: 'your_client_id',
  clientSecret: 'your_client_secret'
});
```
